# Cantina Escolar - Banco de Dados

## Objetivo do trabalho
Desenvolver um banco de dados relacional para uma cantina escolar, contendo clientes, produtos, vendas e itens de venda, utilizando comandos SQL de criação, inserção, atualização, exclusão e consultas com JOIN.

## Ferramenta utilizada
- SQLite Online (modo DuckDB)
- Linguagem SQL

## Como executar o script
1. Acesse o site https://sqliteonline.com
2. Selecione o banco DuckDB
3. Copie todo o conteúdo do arquivo `recuperacao_cantina.sql`
4. Cole no editor SQL
5. Execute o script completo
